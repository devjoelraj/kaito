# Kaito

## Overview

This repository contains the source code for the Kaito project, which is structured into two main parts:

- `cilent/`: Client-side application code.
- `server/`: Server-side application code.

## Tech Stack

- React (Frontend)
- Node.js + Express (Backend)
- MongoDB Atlas

## How to Run Locally

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm start
